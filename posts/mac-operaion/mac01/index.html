【Mac-rar文件解压】Mac系统如何打开rar压缩包文件_无需下载应用程序的快捷4步操作法

0.查阅说明#

大家好👋，因为本人属实很不喜欢下载很多应用程序在电脑上，因此在网络上找到了另外一种解决解压rar压缩包的方法（但是网络上的方法并不是完全适用，因此根据我的电脑做了最新的更新），超快哈哈哈！请注意⚠️该方法目前只适用于Macbook IOS系统，你只需要打开电脑==终端==即可！然后按照我以下超级超级详细的步骤来就可以！努力实现喂饭教学🍚

1. 打开终端#

使用Command+空格键然后搜索终端就可以快速打开

2.输入命令#

输入以下命令并且提示成功下载

1

brew install --cask lunar

请注意⚠️因为在前期的Git实操文章中我已经下载了brew,因此我这里直接可以使用，如果没有下载的朋友可以考虑官网搜索下载，也十分便捷。

//截图如下👇

3. 输入命令#

输入以下命令并提示运行成功，有☑️的符号提示

1

brew install unar

//截图如下👇

4. 开始解压！#

输入以下命令并提示运行成功，解压成功会有提示Successfully，到这里你就已经完成了所有的工作并且解压完成，压缩文件直接放在来你的根目录里。

1

unar 文件路径

🎉以后再要解压就可以直接在终端输入：==unar+你要解压的文件路径名==即可！🎉
